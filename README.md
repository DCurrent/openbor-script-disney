# DC Disney

## Dependencies

## Installation
1. Install any listed dependencies. See an individual dependency's readme for specific instructions.
1. Download and unzip the [latest release](../../releases).
1. Place the *dc_disney* folder into your *data/scripts* folder.
1. Add ```#include data/scripts/dc_disney/main.c``` into any other script you would like to add this libraryâ€™s functionality to. Note any of the "DC" library series listing this as a dependency will already have this step completed.
1. Open *config.h* to modify default values used in the library.
